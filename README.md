#  Python Turtle Graphics – Colorful Flower Design

This project creates a beautiful multi-color floral pattern using **Python Turtle Graphics**.  
The design is generated using loops, angle rotations, and color transitions to produce a symmetric, visually appealing output.

##  Technologies Used
- Python
- Turtle Graphics
- Jupyter Notebook

##  Concept
The pattern is created by:
- Repeating shapes using loops
- Rotating the turtle by specific angles
- Applying gradual color changes (rainbow effect)
- Layering multiple patterns to form a flower-like structure
